# pg001 — テキスト大文字変換ツール

## 概要
標準入力またはファイルのテキストを大文字に変換するCLIツール。

## 使い方
```
pg001 [FILE...]         # ファイル指定（複数可）
echo "hello" | pg001   # stdin から読み込み
pg001 --help           # ヘルプ表示
```

## 設計方針
- `optparse-applicative` で引数を宣言的に定義
- 将来的にオプション（例: `--inplace`, `--encoding`）を追加しやすい構造

## 拡張例（将来）
- `--inplace` / `-i` : ファイルをその場で書き換え
- `--lower` / `-l`  : 大文字→小文字の逆変換
- `--encoding`      : 入力エンコーディング指定
