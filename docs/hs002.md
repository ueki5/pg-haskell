# hs002 — Haskell入門パッケージ

## 概要

Haskell学習の出発点として作成したシンプルなパッケージ。固定の挨拶文を標準出力に表示する。

## 構成

| ファイル | 役割 |
|---|---|
| `src/Lib.hs` | 挨拶文を提供するライブラリ |
| `app/Main.hs` | CLIエントリポイント |
| `test/Spec.hs` | hspecユニットテスト |

## ライブラリ (`Lib`)

```haskell
greeting :: String
greeting = "Hello, Haskell!"
```

定数値として挨拶文を公開するだけのシンプルなモジュール。

## 実行方法

```sh
hs002
# 出力: Hello, Haskell!
```

引数・オプションなし。

## 依存ライブラリ

- `base` のみ（外部依存なし）

## テスト

hspec で1ケース:

| テスト | 内容 |
|---|---|
| 正しい挨拶文を返す | `greeting == "Hello, Haskell!"` |
