# hs003 — ソースファイル読み込み・表示ツール

## 概要

指定したファイルの内容を読み込んで標準出力に表示するCLIツール。コンパイラ実装の足がかりとして設計されており、`Compiler` モジュールがソース読み込みの基盤を担う。

## 構成

| ファイル | 役割 |
|---|---|
| `src/Compiler.hs` | ソースファイル読み込みライブラリ |
| `app/Main.hs` | CLIエントリポイント |
| `test/Spec.hs` | hspecユニットテスト |

## ライブラリ (`Compiler`)

```haskell
readSource :: FilePath -> IO String
readSource = readFile
```

`readFile` のエイリアス。将来的にトークナイズや構文解析などの処理を追加することを想定したインターフェース。

## 実行方法

```sh
# ファイルを指定して内容を表示
hs003 FILE

# ヘルプ表示
hs003 --help
```

ファイルパスは必須引数。内容はそのまま標準出力に書き出す。

## 依存ライブラリ

- `optparse-applicative` — CLIオプション定義

## テスト

hspec で2ケース（実際のファイルI/Oを伴う結合テスト）:

| テスト | 内容 |
|---|---|
| ファイルの内容をそのまま返す | `/tmp/hs003_test.txt` に書き込み後、読み込んで一致確認 |
| 空ファイルは空文字列を返す | 空ファイルを読み込んで `""` と一致確認 |
