# hs004 — 式言語コンパイラ

## 概要

指定したソースファイルを読み込み、字句解析・構文解析・評価を行う式言語コンパイラ。
hs003をベースに作成。外部ライブラリを使用せず `base` のみで実装している。

## 構成

| ファイル | 役割 |
|---|---|
| `src/Compiler.hs` | 字句解析・構文解析・評価の全ロジック（ライブラリ） |
| `app/Main.hs` | CLIエントリポイント |
| `test/Spec.hs` | hspecユニットテスト |

## 実行方法

```sh
# ソースファイルを指定して実行
hs004 FILE

# ヘルプ表示
hs004 --help
```

ファイルパスは必須引数。出力はASTと評価結果の2行。

```
AST: Add (Lit 1) (Mul (Lit 2) (Sub (Lit 3) (Lit 4)))
-1
```

## 言語仕様

- 値は符号付き整数のみ
- 演算は加(`+`)、減(`-`)、乗(`*`)、除(`/`)のみ
- 括弧で演算の優先度を制御
- 言語機能は式のみ（文・ステートメントなし）
- 即値・演算子・括弧で式を構成する

## ライブラリ (`Compiler`) の設計

### トークン型

```haskell
data Token = TInt Int | TPlus | TMinus | TStar | TSlash | TLParen | TRParen
  deriving (Show, Eq)
```

### AST型

```haskell
data Expr
  = Lit Int
  | Add Expr Expr
  | Sub Expr Expr
  | Mul Expr Expr
  | Div Expr Expr
  | Neg Expr
  deriving (Show, Eq)
```

### 公開関数

```haskell
tokenize :: String -> Either String [Token]
parse    :: [Token] -> Either String Expr
eval     :: Expr -> Either String Int
```

### 文法（再帰下降パーサ）

演算子優先度を正しく処理するため、以下の文法で実装:

```
expr   ::= term   (('+' | '-') term)*
term   ::= factor (('*' | '/') factor)*
factor ::= INT | '(' expr ')' | '-' factor
```

- `tokenize`: 空白スキップ、数字列を `TInt` に変換、未知文字はエラー
- `parse`: 上記文法に従い再帰下降で解析、残余トークンがあればエラー
- `eval`: `Div` でゼロ除算を検知して `Left "division by zero"`

## 依存ライブラリ

- `base` のみ（`Data.Char` の `isDigit`, `isSpace` を使用）
- CLIに `optparse-applicative`
