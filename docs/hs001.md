# hs001 — テキスト大文字変換ツール

## 概要

標準入力またはファイルのテキストを大文字に変換するCLIツール。

## 構成

| ファイル | 役割 |
|---|---|
| `src/UpperCase.hs` | 変換ロジック（ライブラリ） |
| `app/Main.hs` | CLIエントリポイント |
| `test/Spec.hs` | hspecユニットテスト |

## ライブラリ (`UpperCase`)

```haskell
uppercaseText :: String -> String
uppercaseText = map toUpper
```

`Data.Char.toUpper` を各文字に適用するシンプルな実装。

## 実行方法

```sh
# ファイル指定（複数可）
hs001 FILE...

# 標準入力から読み込み
echo "hello" | hs001

# ヘルプ表示
hs001 --help
```

引数なしの場合は標準入力を読み込み、ファイルを指定した場合は各ファイルを順に処理して結果を標準出力に書き出す。

## 依存ライブラリ

- `optparse-applicative` — CLIオプション定義
- `attoparsec`, `text` — 将来的な拡張のために導入済み（現時点では未使用）

## テスト

hspec で4ケース:

| テスト | 内容 |
|---|---|
| 小文字を大文字に変換する | `"hello"` → `"HELLO"` |
| 空文字列はそのまま | `""` → `""` |
| 大文字はそのまま | `"ABC"` → `"ABC"` |
| 混在文字列 | `"Hello World"` → `"HELLO WORLD"` |

## 将来的な拡張案

- `--inplace` / `-i` : ファイルをその場で書き換え
- `--lower` / `-l` : 大文字→小文字の逆変換
- `--encoding` : 入力エンコーディング指定
